USE CASES

先確認工作內容，再決定 GPU。

相同 GPU 型號不代表整體效能相同，CPU、RAM、儲存與網路也會影響實際使用體驗。
01

YOLO / AOI 視覺模型

資料前處理、物件偵測訓練、驗證與模型匯出。

常見需求：24GB 以上 VRAM
02

ComfyUI / 圖像生成

Stable Diffusion、工作流測試、批次出圖與模型載入。

常見需求：16–32GB VRAM
03

LLM / RAG / 推論服務

開源模型測試、量化推論、內部知識庫與 API 服務。

依模型大小評估 VRAM
04

Jupyter / 科學運算

CUDA、PyTorch、TensorFlow、資料分析與工程驗證。

需同步確認 CPU、RAM、SSD
