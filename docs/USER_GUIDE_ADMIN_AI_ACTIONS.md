# Admin AI Actions User Guide

## Purpose

Admin AI Actions turn a Lead into practical operating output.

## Buttons

### Analyze Lead

Classifies the Lead and returns:

- Priority
- Score
- Business Unit
- Product / Service
- Missing Information
- Next Action

### Generate Proposal

Creates a Markdown proposal draft.

Use this when the Lead is clear enough for a preliminary proposal.

### Generate Missing Info Email

Creates an email draft asking the customer to provide missing information.

Use this when Lead Intake shows missing required or useful fields.

### Generate Proposal Email

Creates an email draft for sending the preliminary proposal.

Use this after Proposal has been reviewed.

## Output Controls

### Copy

Copies the current output.

### Download TXT

Downloads the current output as a `.txt` file.

### Clear

Clears the saved output for the current Lead.

## Persistence

Admin AI output is saved locally in the browser by Lead ID.

Refreshing the page may keep the output if the same browser and Lead ID are used.

This is not a server-side database.
