這是 HengGou AI Platform 第一個正式公開版。

### Includes

- AI Platform Landing Page
- AI Solutions
- AI Cloud Beta Waiting List
- Case Study Page
- Waiting List AJAX frontend
- Google Apps Script Web API
- Google Sheets CRM
- Gmail notification
- Telegram notification
- Admin Dashboard
- Leads Table
- CRM Pipeline
- Analytics Charts
- API Connection Settings
- Lead Search / Filter / Status Update
- Lead Detail Drawer
- Follow-up Date / Internal Notes
- CSV Export
- SEO metadata
- favicon
- manifest
- robots.txt
- sitemap.xml
- JSON-LD

### Deployment Required

正式上線前請確認：

1. Google Apps Script Web App 已部署。
2. `js/config.js` 已填入正式 API_BASE 與 API_KEY。
3. `/admin/` 已測試 API 連線。
4. Waiting List 實際送出成功。
5. Gmail / Telegram 通知成功。
