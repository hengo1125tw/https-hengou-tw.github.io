# Productization Checklist

## Required Before Customer-Facing Use

- [ ] Public website content reviewed.
- [ ] Admin login tested.
- [ ] Google Apps Script endpoint tested.
- [ ] Waiting List submission tested.
- [ ] Admin lead loading tested.
- [ ] Lead drawer tested.
- [ ] Analyze Lead tested.
- [ ] Generate Proposal tested.
- [ ] Generate Missing Info Email tested.
- [ ] Generate Proposal Email tested.
- [ ] Copy output tested.
- [ ] Download TXT tested.
- [ ] Clear output tested.
- [ ] Mobile layout checked.
- [ ] OpenRouter keys are not stored in browser code.
- [ ] Gmail sending remains manual / explicit.
- [ ] Sensitive customer data is not committed to GitHub.

## Release Rule

If a feature affects email sending, payment, legal claims, warranty, or customer commitments, it cannot auto-send or auto-publish without explicit user confirmation.
